More L-System Plant Info

Our 3D L-system add-on is a Blender add-on that procedurally creates 3D plant-like structures using L-systems and a 3D implementation turtle graphics. The user can define the axiom (starting string), a set of variables (symbols that cause forward drawing movement like F, A, B), and constants (symbols that perform transformations or control branching but do not draw). The constants include: + and - for rotating (left/right turns around the Z-axis), & and ^ for more movement (tilts up/down around the X-axis), \ and / for rolling (rotations around the Y-axis), | to spin 180° around the Z-axis, and [ and ] to use as grouping symbols for the turtle’s position and orientation, this lets us branch like tree limbs.

The rewrite rules for F, A, and B construct how the symbols expand with each iteration, allowing increasingly complex structures. The angle sets the rotation amount for each turning symbol, step length defines how far the turtle moves forward per drawing symbol, and bevel depth controls the thickness of the resulting 3D curve. When the user clicks “Generate L-System,” the add-on recursively rewrites the axiom using the rules, then interprets the resulting string using 3D turtle logic. It tracks movement with a stack for branching, builds separate polylines for each branch, and generates a Blender curve object with beveled splines representing the full structure. The user interface in the 3D View sidebar allows easy control of all parameters, making it ideal for designing trees, vines, fractals, and organic forms procedurally.
